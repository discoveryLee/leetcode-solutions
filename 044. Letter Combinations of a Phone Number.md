# 电话号码的字母组合
给定一个仅包含数字 2-9 的字符串，返回所有它能表示的字母组合。  
给出数字到字母的映射如下（与电话按键相同）。注意 1 不对应任何字母。  
![image](https://upload.wikimedia.org/wikipedia/commons/thumb/7/73/Telephone-keypad2.svg/200px-Telephone-keypad2.svg.png)  
示例:

    输入："23"
    输出：["ad", "ae", "af", "bd", "be", "bf", "cd", "ce", "cf"].
说明:
尽管上面的答案是按字典序排列的，但是你可以任意选择答案输出的顺序。  
我的思路：  
遍历数字字符串，对所对应的字符串集合求笛卡尔积  
代码如下：
```
class Solution {
    public List<String> letterCombinations(String digits) {
        //连续调用组合函数求笛卡尔积
        Map<Character,List<String>> map=new HashMap<Character,List<String>> ();
        List<String> l2=new ArrayList<String>();
        List<String> content2=Arrays.asList("a","b","c");
        l2.addAll(content2);
        
        List<String> l3=new ArrayList<String>();
        List<String> content3=Arrays.asList("d","e","f");
        l3.addAll(content3);
        
        List<String> l4=new ArrayList<String>();
        List<String> content4=Arrays.asList("g","h","i");
        l4.addAll(content4);
        
        List<String> l5=new ArrayList<String>();
        List<String> content5=Arrays.asList("j","k","l");
        l5.addAll(content5);
        
        List<String> l6=new ArrayList<String>();
        List<String> content6=Arrays.asList("m","n","o");
        l6.addAll(content6);
        
        List<String> l7=new ArrayList<String>();
        List<String> content7=Arrays.asList("p","q","r","s");
        l7.addAll(content7);
        
        List<String> l8=new ArrayList<String>();
        List<String> content8=Arrays.asList("t","u","v");
        l8.addAll(content8);
        
        List<String> l9=new ArrayList<String>();
        List<String> content9=Arrays.asList("w","x","y","z");
        l9.addAll(content9);
        
        map.put('2',l2);
        map.put('3',l3);
        map.put('4',l4);
        map.put('5',l5);
        map.put('6',l6);
        map.put('7',l7);
        map.put('8',l8);
        map.put('9',l9);
        
        List<String> list=new ArrayList<String>();
        if(digits.length()==0)
            return list;
        list=map.get(digits.charAt(0));
        int i=1;
        while(i<digits.length()){
           list=combine(list,map.get(digits.charAt(i++)));
        }
        return list;
    }
    //组合函数
    public List<String> combine(List<String> list1,List<String> list2){
        List<String> result=new ArrayList<String>();
        for(int i=0;i<list1.size();i++){
            for(int j=0;j<list2.size();j++){
               result.add(list1.get(i)+list2.get(j));
            }
        }
        return result;
    }
}
```
